# Day 5 – Networking Fundamentals (TryHackMe)

Today I started the "What is Networking" module, focusing on how devices communicate and the rules that govern the internet.

### Key things I learned:
* **IP Addresses:** Learned the difference between IPv4 and IPv6, and how these unique identifiers allow data to find its destination.
* **MAC Addresses:** Learned about the unique physical address every network interface has and how it differs from a logical IP address.
* **The Ping Command:** Learned how to use `ping` to test the reachability of a host on an Internet Protocol (IP) network.
* **ICMP Protocol:** Explored how "Echo Requests" and "Echo Replies" work to verify if a server or website is "up" or "down."

Starting to see how these identifiers and tools are the first step in mapping out a target network.
