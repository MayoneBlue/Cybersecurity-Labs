# Day 3 – Defensive Security Intro (TryHackMe)

Today I learned the basics of Defensive Security and how SOC analysts protect environments.

### Key things I learned:
* **Digital Forensics:** Analyzing footprints left behind by attackers.
* **Threat Intelligence:** Learning about potential adversaries and their motives.
* **Log Analysis:** How to monitor traffic to identify suspicious activity.
* **Malicious URL Discovery:** Investigating an attack where an adversary tried to find hidden directories.

This showed me the "Blue Team" side of security and how to identify an attack in real-time.
